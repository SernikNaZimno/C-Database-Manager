# C++ Database manager #
This porgram in c++ is aimed to help you perform basic operations on your local databases.
It (will) features creating & deleting database as well as creating & removing a table as well 
as adding and removing data from said tables. It (will) features a basic database explorer.
It (will) supports 4 types of database architecture: SQLite, MySQL, Postgres and MangoDB.
<br><br>
In the future it will include connecting to databases remotely from another device.
